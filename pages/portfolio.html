Guns, Germs and Steel

Unity FPS Project · Game Client Developer

프로젝트 개요

Unity 기반 FPS 싱글 플레이 프로젝트로, 전투 감각과 유지보수 가능한 클라이언트 구조를 동시에 고려하여 개발하였다.

담당 역할

  • 게임 클라이언트 전반 구조 설계 및 구현
  • 사격·반동·조준 기반 전투 시스템 개발
  • Procedural Motion 기반 전투 연출 시스템 설계
  • UI–로직 의존성 분리 구조 설계
  • Addressables 기반 리소스 관리

플레이 영상

핵심 시스템 설계

Procedural Motion 기반 전투 연출

키프레임 애니메이션을 기반으로 유지하되, 사격·조준·이동 상태에 따라 Runtime 계산된 Motion을 합성하는 하이브리드 애니메이션 구조를 설계하였다. 


Input
 → Character Controller
 → Weapon / Combat Logic
 → Camera Procedural Motion

UI – 설정 시스템 분리

UI는 이벤트만 발생시키고, 실제 설정 적용은 SettingManager에서 담당하도록 분리하였다. 이를 통해 UI 구조 변경 시 로직 수정이 최소화되었다.

시스템 구조 다이어그램

코드 예시


void ApplyRecoil(float amount)
{
    currentRecoil += amount;
    currentRecoil = Mathf.Clamp(currentRecoil, 0f, maxRecoil);

    cameraTransform.localRotation *=
        Quaternion.Euler(-currentRecoil, 0f, 0f);
}
